# Mercedes-Benz Web Page

Este proyecto es una página web estática inspirada en la marca Mercedes-Benz, desarrollada con HTML y CSS, el sitio presenta información básica de la empresa, sus modelos destacados y un formulario de contacto

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript básico



## Descripción y uso de la actividad 3


Abrir el proyecto en visual studio, despues, los archivos que estan en carpetas, deben estar en el orden del GitHub, despues, correr en la terminal node server.js, cuando salgan mensajes de confirmacion , ej: "Servidor corriendo en puerto 3000, MongoDB conectado" ahora si se podra correr la pagina entera
